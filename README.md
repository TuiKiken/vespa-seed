# Requirements

- [NodeJS with NPM](https://nodejs.org/)

# Install

Just run this console commands in the project directory:

``` shell
npm install
```

# Development

Start development using:

```shell
npm start
```

Run tests with:

```shell
npm test
```

Also you can check your code for accordance with style guides:

```shell
npm run lint
```
